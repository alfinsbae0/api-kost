## Simple API Kost.
Building with
Node Express JS Environment.

## Instalation on local
- clone this repo
```bash
git clone https://github.com/alfinsbae0/api-kost.git
```
- open in your clone folder
```bash
cd api-kost
```
- run npm to install dependencies
```node
npm install
```
- start npm dev
```node
npm start
```
